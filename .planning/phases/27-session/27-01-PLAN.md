**MountNotReadyError（per D-06，参考 ssh_ready.go 的 SSHNotReadyError 风格）：**
```go
type MountNotReadyError struct {
    MountPath string
    Timeout   time.Duration
    LastErr   error
}

func (e *MountNotReadyError) Error() string {
    return fmt.Sprintf("挂载 %s 超时（%v）: %v", e.MountPath, e.Timeout, e.LastErr)
}

func (e *MountNotReadyError) Unwrap() error {
    return e.LastErr
}
```

**channelRWC 适配器（per D-01，将 SSH session pipe 适配为 sftp.NewServer 的 io.ReadWriteCloser）：**
```go
type channelRWC struct {
    io.Reader      // session.StdoutPipe() — 读 sshfs 发出的 SFTP 请求
    io.WriteCloser // session.StdinPipe()  — 写 SFTP 响应给 sshfs
}

func (c *channelRWC) Close() error {
    return c.WriteCloser.Close()
}
```

关键 I/O 方向：`Reader = StdoutPipe()`（读取 sshfs 输出的 SFTP 请求），`WriteCloser = StdinPipe()`（向 sshfs stdin 写入 SFTP 响应）。反接会导致协议死锁。

**mountWorkspace 函数（per D-01, D-02, D-04, D-06, D-07, D-08, D-09）：**

签名：`func mountWorkspace(conn *ssh.Client, localDir string) (cleanup func(), err error)`

实现步骤：
1. `conn.NewSession()` 创建 sshfs 专用 session
2. 在 `session.Start()` **之前**调用 `session.StdinPipe()` 和 `session.StdoutPipe()`（时序关键，Start 之后调用会返回错误）
3. `session.Start("sshfs : /workspace -o passive -f")` 启动容器内 sshfs（per D-04，`-o passive` 替代旧名 `-o slave`，`-f` 保持前台防止 daemonize）
4. 创建 `channelRWC{Reader: stdout, WriteCloser: stdin}`
5. `sftp.NewServer(rwc, sftp.WithServerWorkingDirectory(localDir))` 创建 SFTP server（per D-02，根目录为用户 CWD）
6. 在 goroutine 中运行 `server.Serve()`，用 `chan error` 同步生命周期（`sftpDone := make(chan error, 1)`）
7. 调用 `waitForMount(check, 200*time.Millisecond, 10*time.Second)`，check 函数见下方
8. 返回 cleanup 函数：(1) `sshfsSession.Close()`（per D-07，关闭 channel 触发 EOF → sshfs 自动卸载），(2) `<-sftpDone`（等待 SFTP goroutine 结束），(3) `server.Close()`，(4) `fusermountCleanup(conn)`（per D-08）

错误处理：如果步骤 2-7 中任何一步失败，关闭已创建的 session 后返回错误。错误消息使用中文 + `%w` 包装（如 `"创建 sshfs session 失败: %w"`）。

mountpoint 检测 check 函数（传给 waitForMount）：
```go
check := func() error {
    sess, err := conn.NewSession()
    if err != nil {
        return err
    }
    defer sess.Close()
    return sess.Run("mountpoint -q /workspace")
}
```

**waitForMount 函数（per D-06，参考 ssh_ready.go 轮询结构）：**

签名：`func waitForMount(check func() error, interval, timeout time.Duration) error`

接受可注入的 check 函数（便于单元测试）。使用 `time.NewTimer(timeout)` + `time.NewTicker(interval)` + `select` 循环。超时返回 `&MountNotReadyError{MountPath: "/workspace", Timeout: timeout, LastErr: lastErr}`。首次进入循环前先执行一次 check（与 ssh_ready.go 一致）。

**fusermountCleanup 函数（per D-08）：**

签名：`func fusermountCleanup(conn *ssh.Client)`

创建短生命周期 session，执行 `fusermount -u /workspace 2>/dev/null || true`。所有错误静默忽略（此函数为防御性兜底，失败不影响退出码）。
