# Quick Task 260708-q1b: 降低 VNC 与 Chromium 默认资源占用

## Goal

降低 KasmVNC 编码与 Chromium 多进程默认开销，减少高负载下 VNC 会话被资源压力打掉的概率。

## Tasks

1. 用测试锁定 VNC 轻量配置：`max_frame_rate=30`、`max_quality=8`、`rectangle_compress_threads=2`。
2. 用测试锁定 Chromium 默认 renderer 进程数可配置且默认较保守。
3. 更新 `entrypoint.sh` 和 `restart-vnc.sh` 中重复的 KasmVNC 配置。
4. 更新 `launch-chromium.sh` 默认 flags。
5. 验证 Go 测试与 shell 语法。
