## Objective

彻底移除 Cloud CLI Proxy 中的端口映射（Port Mapping）特性，包括后端代码、数据库模型、网络层兼容补丁、API 路由、前端组件及相关文案。

Purpose: 端口映射特性与产品核心约束（全流量必须走隧道，不允许直连外网）相冲突，移除后网络策略更纯粹，维护面更小。
Output: 所有端口映射相关代码、模型、API、UI 和文档文案清理完毕，项目编译和测试通过。
