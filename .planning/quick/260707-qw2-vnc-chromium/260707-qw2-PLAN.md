# Quick Task 260707-qw2: VNC 状态展示、启动重启控制、自动拉起与 Chromium 窗口适配

## Goal

让 VNC 在容器内具备可观测状态、手动启动/重启入口和受限自动拉起能力，并让 Chromium 默认窗口不再超出 VNC 视口。

## Tasks

1. 后端增加 VNC 状态探测与 `start`/`restart` 控制返回，管理员和用户接口保持权限边界。
2. 受管用户镜像脚本增加 VNC watcher，30s 内最多自动拉起 3 次，稳定成功后重置计数。
3. 前端管理员详情页和用户门户详情页展示 VNC 状态，并根据能力启用启动/重启按钮。
4. 调整 Chromium 默认窗口尺寸，保留最大化体验但避免 1920x1080 窗口装饰溢出。
5. 运行 Go/前端/脚本验证后提交、推送并打补丁 tag。
